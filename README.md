# Scitriad Games

Welcome to the **Scitriad Games** repository!

Scitriad Games is a collection of interactive educational games designed specifically for **Alberta high school students**. The activities align with Alberta’s secondary curriculum in **Science, Mathematics, and Humanities**, helping students strengthen their understanding through engaging, game-based learning.

Whether you're reviewing for a diploma exam, reinforcing classroom concepts, or exploring topics more deeply, Scitriad Games offers interactive tools to support your success.

---

## 🎓 Built for Alberta High Schools

This project focuses on:

* Alberta Science courses (Biology, Chemistry, Physics, General Science)
* Alberta Mathematics courses (Math 10–12 streams)
* Social Studies and Humanities content relevant to Alberta curriculum outcomes
* Skills development aligned with provincial standards

Teachers can use these games as classroom supplements, review activities, or enrichment tools. Students can use them independently for practice and exam preparation.
